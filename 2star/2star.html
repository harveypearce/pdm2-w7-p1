Environments

Create an Environment

In the second Lecture we also learnt about how we can create environments using Tile sets

Start by finding a Tile Set you want to use (opengameart, itch.io )

  1. First you must add the sprite sheet as an asset and load it onto your sketch using preload()
  2. Using the extra parameters of image() to crop the image to each tile
  3. Place the tiles to create an environment.

Using Classes to Create an Environment

Now we have put everything together lets use create a class to help speed up making environments.

  1. Create a new .js file in your project. name it 'enviroment.js'
  2. In this file create a new class called 'EnvironmentTiles'
  3. In the class create a method for each of the tiles you want to use.

To complete this task think about how layers work in p5.js. Is it possible for the player to talk behind an object?
