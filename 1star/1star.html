Create a Sprite

  1. Use preload() to load in the sprite sheeting in the assets folder.
  2. Animate your character using the JSON and sprite sheet example in the lecture. Look at W7L1_Animation_Single_SpriteSheet_Simple for an example of this method.

Move the Sprite

Now that the character is animated, let's make it move.

  1. Use keyPressed() or keyIsDown() to make the sprite move around the canvas (don't worry about changing animations yet)

Now that the character can move, you should notice two issues. First, the sprite does not change. Second, the animations only match some movement, e.g. there is only a 'right' moving animation and not a 'left'. Let’s fix this.

  1. Modify your code to use different animations on the sprite sheet. If you are using the example sheet, you will notice that each row is a single animation sequence. (you will need to edit the json, have a go and asked for help)
